ဒီမှာမှာကိုကျော်နိုင်အောင်ဖြောင့်ကြောင့်ရကုန်အံ့
ကျနော်တို့ကိုတက်စေမယ်ပု Raspberry Pi 3
အဆိုပါ Odroid မှာ terminal ကိုဒီမှာကျော်
xu4 တန်းစီငါဂုဏ်နင့်ကိုအတိအကျတူညီပြုပါ
အရာသည်ယခုသင်ဤအပြည့်အဝဖြစ်ပါတယ်သတိထားမိပါလိမ့်မယ်
operating system ကိုဒီကိုကြိုက်သည်မဟုတ်
နှိမ့်ချ-ချွတ်ဒီသစ်ကိုစန်းစန်းဖြစ်ပါသည်
တကသငျသညျဒါကျော်ကဒီမှာမျှော်လင့်ထားလိမ့်မယ်အဖြစ်
ငါဂုဏ်နသွားလာရင်း H ကိုထိပ်ဆုံးပေါ့ကျနော်တို့လေးယောက် cores တွေ့မြင်
ဒါကြောင့် RAM ကို 750 megabytes တူနှင့်အတူ
နှင့်ကျော်ကဒီမှာကျွန်တော့် Odroid xu4 အပေါ်
ရှစ် core နဲ့သိုး၏ 1.9 ငါးခုအရင်ဖျော်ဖြေတယ်
