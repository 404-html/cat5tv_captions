آپ ہاں اور یہ بہت اچھا ہے اور میں ہوں
باقیوں کو سننے کے لئے آپ کو مارنے کے لئے پرجوش
کارمین کی کہانی اور جو سب خدا
کیا ہے اور آپ کو کیا معلوم ہے
اس نے کہا کہ خدا آپ کو ہر نیا کرنا چاہتا ہے
وہ دن آپ کے دماغ اور میں تجدید کرنا چاہتا ہے
محبت اور آپ کو قبول کرتے ہیں اور میں نے تجربہ کیا ہے
وہ میری اپنی زندگی میں اور ہم نے سنا ہے
کارمین اور ہمارے پاس ابھی تک بہت زیادہ ہے
اس موضوع پر آپ کے بارے میں بات کریں تو آپ کو
اگلے قسط کے لئے ہمارے ساتھ ملیں گے
کارمین یہ بہت اچھا ہو گا کہ میں پیار کرتا ہوں
اور ہم اپنے شراکت داروں کا شکریہ ادا کرنا چاہتے ہیں
ایک کمر تیار کریں جو آپ ہمارے پاس جا سکتے ہیں
ویب سائٹ ہر روز ٹی وی آپ پکڑ سکتے ہیں
ویب سائٹ پر اس پرکرن یا آپ کر سکتے ہیں
سلائی پر ہمیں پکڑو لیکن ڈیزائن کریں
ٹی شرٹ یہ ہے کہ آپ اپنی مرضی کے مطابق ڈیزائن کرسکتے ہیں
اگر ٹی شرٹ آپ لنک کو پکڑ سکتے ہیں تو
آپ ہمارے شراکت دار کے صفحے پر جائیں گے
اگر آپ تلاش کر رہے ہیں تو وہاں سے بھی کلک کریں
کچھ مصنوعات کے لئے ایمیزون کے ذریعے ہے
ہمارے صفحے پر ایک لنک ہے کہ آپ خرید سکتے ہیں
مسیحی کتابیں آپ بائبل خرید سکتے ہیں
ایمیزون کے ذریعے بھی اگر آپ کے پاس نہیں ہے
ایک ہماری ویب سائٹ پر شراکت داروں کے پاس جاؤ اور
پھر ایمیزون بٹن پر کلک کریں تو ہم ایسا ہی ہیں
اس ہفتے میں آپ کو محسوس کیا کہ شکر گزار
ہم امید کرتے ہیں کہ آپ اگلے ہفتے ہمارے ساتھ ملیں گے
کیا ہم قریبی طور پر ایک ساتھ دعا کر سکتے ہیں
وہاں کارمین آسانی سے والد صاحب میں چاہتا ہوں
آپ کا بہت اچھا شکریہ
ہمارا فضل اور قبولیت اور محبت
کہ ہم کمائی نہیں کرتے تھے لیکن آپ نے صرف اسے دیا
ہمارے لئے اور ممکن ہو کہ ہم یہ کرسکیں
اپنے بچوں کو یسوع کے ذریعہ بنیں
مسیح اور اس نے صلیب پر کیا کیا
ہمارے گناہوں اور میں لوگوں کو دیکھ رہا ہوں
یہاں یا وہاں خدا آپ کو سنتے ہیں
اس طرح اپنی زندگیوں کو چھونے اور انہیں ڈرا دیں گے
اپنے آپ کو یہ کہ آپ انہیں نیا دے دیں گے
مسیح میں زندگی کارمین اور آپ کا شکریہ
جو کچھ آپ نے بہت اچھا کیا ہے
وہ شخص کہتا ہے کہ ہم آپ کو دعا کرتے ہیں
اس کا بھلا کرے گا ہم کرنے کے شوقین ہیں
آپ نے کیا کیا ہے اس بارے میں مزید سننا
اس کی زندگی ہم یسوع کا نام امین میں دعا کرتا ہے
ہمیں ایک حیرت انگیز ہفتے ملے گا جسے آپ کو امید ہے
